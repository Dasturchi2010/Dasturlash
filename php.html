PHP nima?

PHP "PHP: Hypertext Preprocessor" ni qisqartmasi hisoblanadi. PHP yuqori dasturlash tillari sirasiga kiradigan, ochiq kodli skrip tilidir. PHP skriptlari server tamonidan bajariladi. PHP yuklab olish va undan foydalanish bepul. PHP frameworklar Laravel Symfony CodeIgniter Yii CakePHP Phalcon Laminas FuelPHP PHPixie Slim yuqorida keltirilgan frameworklardan boshqa frameworklar ham mavjud bu ketma-ketlik mashxurlaridan boshlab misol keltirildi. Tanlash o'zingizni qo'lingizda bu frameworklardan foydalanish uchun siz PHP dasturlash tilini bilishingiz zarur bo'ladi.

Nima uchun PHP?

PHP turli platformalarda ishlaydi (Windows, Linux, Unix, Mac OS X va boshqalar). PHP bugungi kunda ishlatiladigan deyarli barcha serverlar (Apache, IIS va boshqalar) bilan mos keladi. PHP keng ko'lamli ma'lumotlar bazalarini qo'llab-quvvatlaydi PHP bepul. Uni rasmiy PHP resursidan yuklab oling: www.php.net
